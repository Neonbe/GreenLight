What does "App is damaged and can't be opened" mean?

When macOS says an app "is damaged and can't be opened," it rarely means the file is actually corrupt. It means macOS Gatekeeper detected the com.apple.quarantine extended attribute on the app — a flag automatically added to all files downloaded from the internet. The app was flagged because it hasn't been notarized (verified) by Apple, not because it contains malware.
